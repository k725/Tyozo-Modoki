============

特徴
--

 - ナウい感じでファイルを消せます。
 - アップロードサーバも兼ねてます。

 必要なものとかがあるので[Wiki][1]読んでください。

[Gyazo+][2]での設定例

    [Gyazo+]
    upload_server=pic.example.com
    upload_path=/upload
